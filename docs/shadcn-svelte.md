# Shadcn Svelte

Shadcn/ui for SvelteKit project.

## Usage

```bash
npx shadcn-svelte@latest add button
```

```svelte
<script lang="ts">
  import { Button } from "$lib/components/ui/button/index.js";
</script>

<Button>Click me</Button>
```
