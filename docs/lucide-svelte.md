# Lucide Svelte

Lucide Icons for SvelteKit project.

## Usage

```svelte
<script>
  import { Skull } from '@lucide/svelte';
</script>

<Skull />
```

Additional props can be passed to adjust the icon:

```svelte
<script>
  import { Camera } from '@lucide/svelte';
</script>

<Camera color="#ff3e98" />
```
